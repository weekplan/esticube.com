# EstiCube — US Construction Cost Calculators

Free, fast construction calculators for the US market. Built for SEO and Core Web Vitals.

## Calculators

- **Fence Calculator** — posts, panels, materials, installed cost
- **Deck Calculator** — boards, joists, footings, railing, build cost
- **Sprinkler & Irrigation Calculator** — zones, heads, pipe, water flow

## Stack

- Next.js 16 (App Router, static generation)
- TypeScript + Tailwind CSS v4
- System fonts (no external font requests)
- Inline SVG iconography
- JSON-LD structured data (WebSite, WebApplication, FAQPage, BreadcrumbList)

## Performance

- Mobile-first responsive design
- Minimal client JavaScript (calculators only)
- Dynamic import with loading skeleton
- No external images or analytics by default
- `lang="en-US"`, canonical URLs, sitemap, robots.txt

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```
